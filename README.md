# todo-list-mvc-nodejs
Simple task list using mysql mvc struct and nodejs

# run
1- npm install

2- npm start

it shuld open at `localhost:3000`

# stack
* NodeJs
* ExpressJS
* Handlebars
* MVC
* Sequelize (ORM)
* Nodemon
* MySQL
